Enkozesa ya ebiseela

1. Okufuna ebiseela

var leelo = $.ebiseelaNeEssaawa.olwaleelo({

}); //

2. Okufuna ebiseela mu bigambo byoka

var leelo = $.ebiseelaNeEssaawa.olwaleelo({

}); //

3. Okufuna ebiseela mu bufunze nga oyawuza akasaze

var leelo = $.ebiseelaNeEssaawa.olwaleelo({
endabika: 'kbn',
akagata = '-';
}); //
